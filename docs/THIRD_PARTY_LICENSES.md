# Third-party license inventory

The authoritative release inventory is generated from the exact `package-lock.json` used for the
build:

```bash
npm run release:metadata
```

Every GitHub prerelease attaches:

- `charter-<version>.spdx.json` — SPDX SBOM generated by npm;
- `third-party-licenses.json` — deduplicated package/version/license records;
- `THIRD_PARTY_NOTICES.md` — human-readable inventory;
- `SHA256SUMS.txt` — digests for the artifacts and metadata above.

The generated inventory includes both runtime and development dependencies and labels their scope.
It is an index, not a replacement for the complete license text shipped in or linked by each package.
The installed package manifest and upstream source remain authoritative when an entry is ambiguous.
