name: cdp-hidden-tab-dialog-freeze type: reference origin: db3bf914…00e8 CDP 后台/遮挡标签页里 Chrome 冻结 rAF 与 CSS 动画，Base UI Dialog 永远关不掉；验证对话框类交互必须用前台标签。

用 [[web-access-cdp-setup]] 的代理开的新标签是后台标签（document.hidden=true）；Chrome 会冻结遮挡标签页里的 rAF 与 CSS 动画。任何页面的对话框在隐藏标签里都无法关闭（取消/X/Esc/CDP 可信点击/禁动画注入全都无效，popup 弹窗 + pre-change 代码证明与业务代码无关）。

How to apply:

  • 浏览器验证「对话框关闭/过渡动画类交互」必须用前台可见标签（用户在场时用他们正开着的标签）。
  • 隐藏标签里只验证：渲染内容、console 错误、网络请求/响应、DB 副作用——这些不受冻结影响；不要在隐藏标签里断言「对话框已关闭」。
  • 如果必须自动化对话框交互，先 Page.bringToFront 把标签带到前台再操作。
