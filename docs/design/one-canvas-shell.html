Charter — 一个画布

只有一个工作面，从打开到关闭都是它：侧栏（任务 + 一棵文件树）· 中列（文件画布）· 右列（agent 会话）。 Home 溶解成画布的空态；Task Room 溶解成右列的会话 tab；"编辑器"这个词从产品里消失——中列本来就是编辑器。 没有 surface 切换，没有 ⌘E，没有"回到 Home"。驱动仍然可插拔（◆ 内置 / ✳ claude / ▸ codex / $ 终端）， 监督底座（快照 · 记账 · 字节级回滚 · 验收门）原样保留，长在这一个画布上。
