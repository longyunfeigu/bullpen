这份 mock 的组成： 位置生命周期来自「方向 3 · 检测升格」（shell 在底部坞 → 检测到 claude/codex 进入交互态 → 同一 PTY 升到右侧竖栏 → 退出归位）； 右栏自带「本次会话改动」条（FS watcher 驱动，对 CLI 零解析）；点改动行开 live-follow peek（「方向 B」，即 ADR-0014 peek 的常开跟随形态）； 「⤢ 升级为房间」把外部会话提升为一等 Task Room（「方向 A」：任务列表可见、Changes rail、REVIEW_READY、不自动 ACCEPTED）； 安全底座贯穿全程：进入交互态即 EXT 徽章 + 自动快照 snap-142，收尾提供 Review 与一键回滚。 首次升格右栏应弹询问（总是/从不），本演示按「总是」预设播放。
