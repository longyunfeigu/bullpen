Sourcegraph 把 Prompt Library 做成一等公民;Android Studio 的 prompt library 支持模板变量并随项目 git 共享;社区大量「把 prompt 当代码资产管理」实践。但没有工具把「模型×力度×信任×文件域×验证命令」这套发射参数一起模板化——这正是 Charter composer 的既有维度。
